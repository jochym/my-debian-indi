INDI APT Repository (unofficial)

Community-maintained repository for INDI Library and 3rd-party drivers, built for Debian-based distributions (Trixie/Stable). Automatically built from upstream tags at indilib/indi and indilib/indi-3rdparty.

Note: This is an independent, unofficial repository with no affiliation to the astroberry project or the upstream INDI organization.

GPG Key

Public key for verifying repository signatures:

Download public key (ASCII armor)

curl -fsSL https://jochym.github.io/debian-indi/gpg_public_key.asc | sudo gpg --dearmor -o /etc/apt/trusted.gpg.d/debian-indi.gpg

Adding the Repository

Add to /etc/apt/sources.list.d/debian-indi.list:

deb [signed-by=/etc/apt/trusted.gpg.d/debian-indi.gpg] https://jochym.github.io/debian-indi/ stable main

Or without GPG verification (faster for testing):

deb [trusted=yes] https://jochym.github.io/debian-indi/ stable main

Then:

sudo apt update
sudo apt install indi-3rdparty-drivers

Available Packages

  • indi-bin, libindi1, libindi-dev, libindi-data – core INDI Library
  • indi-3rdparty-libs – meta-package for 3rd-party libraries (empty)
  • indi-3rdparty-drivers – meta-package for all 3rd-party drivers (Recommends all indi-*)
  • Several dozen drivers: indi-eqmod, indi-gphoto, indi-asi, indi-qhy, indi-sbig, indi-playerone and more

Individual packages can be downloaded from the GitHub Releases page.

Architectures

amd64 (x86_64) and arm64 (aarch64 – Raspberry Pi, Orange Pi, etc.)

Updates

Repository is refreshed manually when upstream tags are released. Versions are always synchronized with indilib/indi and indilib/indi-3rdparty.
